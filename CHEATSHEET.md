
# Cheat sheets
## Bash
`C-r` | fzf history | Built-in
`C-t` | Transpose chars | Custom (Replaces default C-n)
`C-n` | fzf filename to yank | Custom
`M-c` | fzf directors to change to | Built in
`C-G` | fzf directors to change to | Custom (D)
`M-.` | Cycle yank through previous args | Built in 
`C-p` | fzf files to open in nvim | Custom
`M-<BS>` | Delete word backwards | Built in
`C-y` | Yank word | Built in
`S-<BS>` | Delete line backwards | Built in
`C-U` | Delete line backwards | Built in
`C-d` | Delete letter forward | Built in
`M-d` | Delete word forward | Built in

## Tmux
`prefix d` | Detach from session
`prefix J` | List sessions to jump to
`prefix K` | List dirs in HOME and workspace to start a new session

`prefix k` / `prefix C-k` | Previous window
`prefix j` / `prefix C-j` | Previous window
`prefix v` | Vertical split
`prefix n` | Horizontal split
`prefix x` | Close pane
`prefix c` | New window
`prefix X` | Close window

`prefix b` | Break current pane into a new window
`prefix C-b` | Break current pane into a session
`prefix B` | Move pain via choose-tree
`prefix m` | Join pane from window
`prefix M` | Join pane from choose tree
`prefix s` | Swap pane 
`prefix S` | Swap pane via choose-tree
`prefix w` | Choose tree

`prefix l` / `prefix C-l` | Last window
`prefix L` | Last session
`prefix ;` / `prefix C-;` | Last pane
`prefix '` / `prefix C-'` | Copy mode
`C-h/k/k/l` | Navigation
`M-h/k/k/l` | Resizing
`prefix i` | Synchronize panes


Layout
`prefix z` | List dirs in workspace to start a new session
`prefix q` | Show pane numbers
`prefix y` | Swap panes
`prefix <Space>` | Next layout
`prefix M-<1-5>` | Choose a layout (3 is large on on the top)
`prefix {` | Move pane left
`prefix }` | Move pane right
